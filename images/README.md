# Images

Place your photos here. File names must match the paths in `content.js`.

| File              | Used in                       |
|-------------------|-------------------------------|
| `hero.jpg`        | Hero background               |
| `story1.jpg`      | Story milestone 1             |
| `story2.jpg`      | Story milestone 2             |
| `story3.jpg`      | Story milestone 3             |
| `gallery1.jpg`    | Gallery                       |
| `gallery2.jpg`    | Gallery                       |
| …                 | Gallery (add as many as you like) |

**Tips:**
- Recommended formats: JPEG or WebP.
- Hero image: at least 1920×1080 px.
- Gallery images: 800×800 px square crops work well.
- Compress images before uploading (e.g. [Squoosh](https://squoosh.app/)).
